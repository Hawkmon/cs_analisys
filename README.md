Анализ статистики профессиональных игроков CS2.

Программа для анализа статистики игроков в Counter-Strike по данным из Excel-файла.

Возможности:
- Анализ статистики игрока (K/D, ADR, Rating, KAST, и другие метрики)
- Фильтрация по временному периоду (месяц, 3 месяца, полгода, год, произвольный интервал)
- Фильтрация по конкретному турниру и стадии
- Фильтрация по карте и оппоненту
- Визуализация динамики показателей
- Автоматический расчет производных метрик

Для выбора опций исползуйте числа, представленные программой.

Используемые сокращения:
gs - групповая стадия
po - плей-офф
k — убийства
d — смерти
adr — средний урон за раунд
rating — рейтинг (HLTV)
kast — процент раундов с убийством, ассистом, спасением или торговлей
swing — показатель реального влияния действия игрока на победу в раунде
kd — отношение убийств к смертям
kpr — убийств за раунд
dpr — смертей за раунд
surv% — процент выживаемости
